使用notepad++批量在每行(每列)首尾添加內容

資料來源: https://blog.csdn.net/lk142500/article/details/83119029

(每列)開頭添加內容
    01.將編輯游標移動到第一列
    02.Tool -> 編輯 -> 列模式編輯...(Alt+C) -> 跳出的UI在「插入文字」的文字框 輸入對應

(每列)尾巴添加內容
    01.Tool -> 搜尋 -> 尋找 -> 跳出的UI切到「取代頁面」在「尋找內容」的文字框 輸入「$」表示每列結尾
	
(每列)替換 換行符號
    01.Tool -> 搜尋 -> 尋找 -> 跳出的UI切到「取代頁面」在「尋找內容」的文字框 輸入「\n」，勾選「規則運算式(正則表達式)」
